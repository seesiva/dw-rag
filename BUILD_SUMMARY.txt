1. Install dependencies:
   pip install sqlalchemy pandas pymysql psycopg2-binary

2. Create PostgreSQL database:
   createdb dw_rag

3. Extract raw data:
   cd src && python extract.py

================================================================================
DESIGN DECISIONS
================================================================================

1. Submitted records only: Filters all facts to docstatus=1
2. Item attributes in long format: Enables Power BI slicers by attribute
3. Signed net amount: Returns automatically negate, revenue = SUM(signed_amount)
4. Surrogate keys: Avoids multi-column joins, enables SCD later
5. Fiscal year Apr-Mar: Aligns with garment industry
6. Idempotent pipelines: TRUNCATE+INSERT for safe reruns
7. Large table chunking: 10K-row batches for 1.26M stock ledger
